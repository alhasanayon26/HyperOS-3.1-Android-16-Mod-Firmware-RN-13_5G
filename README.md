Gold Images [Malachite Base] HyperOS 3.1 Android 16 (Unofficial)
Redmi Note 13 5G (Gold / Malachite)

A lightweight, optimized and customized HyperOS 3.1 Android 16 ROM based on Project EclipseOS.

This ROM is an unofficial customized build created for the Redmi Note 13 5G (Gold/Malachite). It is based on Project EclipseOS by Harsh10R, but has been rebuilt and further optimized with the goal of providing a cleaner, lighter, smoother and more stable daily driving experience.

Instead of adding unnecessary modifications, this build focuses on reducing system overhead while keeping useful features that improve usability and performance.

Important Notes
The device may feel slightly slow after the first boot. This is completely normal while Android finishes optimization.
Initial boot may take several minutes.
The device may become warm during the first setup.
Allow the ROM approximately 24 hours to fully settle for the best performance and battery life.
If Control Center causes SystemUI to restart the first time, simply open it again. This only happens once.
For Control Center customization:
Open Control Center

Flashing Instructions
Requirements
Bootloader unlocked
Battery above 50%
Official USB cable recommended
Original ROM package
Windows, Linux or macOS
Installation
Boot your device into Fastboot Mode.
Connect the phone to your PC.
Extract the ROM.
Run:
Windows
flash_all.bat
Linux / macOS
flash_all.sh
Wait until flashing finishes.
The device will reboot automatically.
First boot may take several minutes.
Known Issues
NFC may not work on some variants.
If you discover any other issue, please provide proper logs and reproduction steps when reporting it.
Recommendations
Charge the device for at least one hour after installation.
Give the ROM about one day to fully optimize itself.
If you experience heating after several days, you may remove unnecessary MMI applications using ADB if desired.
Read the flashing instructions carefully before proceeding.
